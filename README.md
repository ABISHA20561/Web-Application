Student Enrollment Form:
A clean, responsive HTML-only student registration form designed to capture essential student data and course preferences.
Features:
Data Validation: Uses HTML5 attributes like required, pattern (for blood groups), and specific input types (email, number, date).
User-Friendly UI: Includes placeholders, grouped course selections using <optgroup>, and organized sections with <fieldset>.
Comprehensive Fields:
Personal Details (Name, Reg No, Password, DOB, Gender, Blood Group).
Contact Info (Email, Mobile).
Academic Info (Department & Course selection).
Logistics (Mode of study & Additional facilities).
Technologies Used:
HTML5: Structured markup and native form validation.
